# PDF-line-fixer
Standalone tool to fix broken line breaks when pasting PDF text into Word

## Live Tool
👉 **[Open the tool](https://case-thread.github.io/PDF-line-fixer/pdf-line-fixer.html)**

## How to use

1. Copy text from your PDF (Ctrl+A, Ctrl+C inside the PDF viewer)
2. Paste into the input box
3. Choose **Smart Merge** for automatic detection, or one of the other modes
4. Click **Process** and copy the cleaned output

## Modes

- **Smart Merge** — automatically detects real paragraph breaks using scoring heuristics
- **Manual Markers** — you mark the real breaks yourself with a separator line
- **Single Paragraph** — collapses everything into one block
